# ACC 赛事结果分析

面向 **Assetto Corsa Competizione (ACC)** 的会话结果 JSON 可视化：排行榜、圈速与一致性等（Web 应用）。

## 技术栈

React、TypeScript、Vite、Recharts

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

- **GitHub Pages**：`npm run build:GithubPages`
- **EdgeOne Pages**：`npm run build:edgeonePages`

预览生产构建：`npm run preview`
